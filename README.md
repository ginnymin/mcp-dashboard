# Observability dashboard

A simple full stack application that demonstrates an MCP server observability dashboard.

## Run the app

Requires **Node 22–25** (`.nvmrc` points at 25).

```bash
npm install
npm run seed    # Seeds the database with realistic MCP traffic
npm run dev     # Starts API server (port 3001) and frontend (port 3000)
```
